Magnetic Lasso Tool

To use the Polygon Lasso Tool, right click on the Lasso Tool Icon. Among the 3 options that will appear, select Polygon Lasso Tool. This Lasso Tool option allows you to cut images using straight lines. To select a section of an image to cut

The lines must encircle the section of the image you want. The lines should change from black into an alternating color of black and white. This meant that the selected part is ready to be moved using the Move Tool.
